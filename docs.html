01

Introduction

NGRAPHICS is an AI-powered product infographic generator that creates professional marketing visuals for e-commerce and social media platforms.

Using advanced AI image generation models through OpenRouter, NGRAPHICS transforms your product information into stunning infographics complete with callouts, icons, and professional styling.

AI Image Generation

Generate professional product infographics using state-of-the-art AI models like Gemini, GPT-4o, and Flux.

Marketing Copy

Automatically generate product descriptions, taglines, social media captions, and SEO meta text.

Multiple Styles

Choose from various style presets or customize every aspect of your infographic's appearance.

Platform Presets

One-click presets optimized for Amazon, Shopify, Instagram, Facebook, and more.

02

Quick Start Guide

Get your first infographic generated in under 2 minutes:

1

Enter Your API Key

Scroll to the bottom of the configuration panel and expand "API & Model Settings". Paste your OpenRouter API key and click "Save".

2

Add Product Title

Enter your product name in the "Product Title" field. Be specific and descriptive.

3

Add Product Features

Add 3-6 key features or characteristics. Use the star button to mark your most important features.

4

Upload Product Image (Optional)

Drag and drop or click to upload a product photo. The AI will use this as a reference.

5

Choose a Style

Select a visual style from the style options, or leave on "Auto" for AI-recommended styling.

6

Generate!

Click "Generate Infographic" or press Ctrl+Enter. Your infographic will appear in the output panel.

03

API Setup

NGRAPHICS uses OpenRouter to access various AI image generation models. Here's how to set it up:

Getting an OpenRouter API Key

  1. Visit openrouter.ai and create an account
  2. Navigate to the API Keys section in your dashboard
  3. Create a new API key with appropriate permissions
  4. Copy the key (it starts with sk-or-)

API Key Security

Your API key is stored locally in your browser and never sent to any server other than OpenRouter. It persists across sessions for convenience.

Available AI Models

NGRAPHICS supports multiple AI models, each with different strengths:

Model Best For Speed Quality
Gemini 2.0 Flash Fast iterations, general use Fast Good
GPT-4o Complex layouts, text accuracy Medium Excellent
Flux Photorealistic images Medium Excellent
Riverflow Artistic styles Medium Good
04

Product Information

Product Title

The product title is the main heading that appears on your infographic. For best results:

  • Be specific: "Premium Wireless Noise-Cancelling Headphones" is better than "Headphones"
  • Include the brand name if relevant
  • Keep it concise but descriptive (3-8 words)
  • Match the language setting (English or Romanian)

Language Selection

Toggle between English (EN) and Romanian (RO) in the header. This affects:

  • Generated text on the infographic
  • Marketing copy output
  • Placeholder text and hints
  • Romanian mode uses proper diacritics (ă, â, î, ș, ț)
05

Characteristics & Features

Product characteristics are the key features displayed on your infographic with callouts and icons.

Adding Features

  • Click "Add Characteristic" to add new feature fields
  • Enter specific, benefit-focused features
  • Aim for 3-6 features for optimal layout

Star (Primary) Features

Click the star icon next to any feature to mark it as primary:

  • Starred features appear larger or more prominent
  • They're emphasized in the marketing copy
  • Use for your 1-2 most important selling points

Auto-Icon Suggestions

As you type, NGRAPHICS automatically suggests relevant icons based on keywords:

"battery" → Battery icon
"waterproof" → Water drop icon
"bluetooth" → Bluetooth icon
"camera" → Camera icon

You can also manually select icons using the icon dropdown.

Pro Tip

Write features as benefits, not just specs. Instead of "5000mAh battery", try "5000mAh battery for 30 hours of playback".

Product Image Upload

Uploading a product image helps the AI create more accurate and realistic infographics.

Image Requirements

  • Supported formats: JPG, PNG, WebP
  • Recommended resolution: 1000x1000 pixels or higher
  • Clean background preferred (white or transparent)

Auto-Enhance

Enable "Auto-enhance" to automatically improve your product photo:

  • Adjusts brightness and contrast
  • Applies auto-levels for better color balance
  • Sharpens details slightly

AI Image Analysis

Click "Analyze with AI" to automatically extract:

  • Product title from the image
  • Visible features and characteristics
  • Color palette suggestions

Image Size

Large images are automatically resized to reduce API costs and improve speed. For best results, crop your image to focus on the product before uploading.

Style Reference

Upload a reference image to guide the AI's visual style.

The AI analyzes your reference image and attempts to match:

  • Color palette and tones
  • Visual style and mood
  • Layout patterns
  • Typography style

Style Strength

Seed Control

Seeds allow you to reproduce or vary specific generations.

Random Seed

When enabled (default), each generation uses a random seed, producing unique results every time.

Fixed Seed

Disable random seed and enter a specific number to:

  • Reproduce a previous result
  • Create variations with minor setting changes
  • Share reproducible results with others

Copying Seeds

After generation, the seed is displayed below the image. Click "Copy" to save it for later use.

14

Image Variations

Generate multiple versions of your infographic simultaneously.

How It Works

  • Select 1, 2, or 4 variations in Advanced Options
  • Each variation uses a sequential seed (base seed + 0, 1, 2, 3)
  • All variations are generated in parallel for speed
  • Click any variation to select it as the primary image
  • Double-click to open in lightbox

Pro Tip

Generate 4 variations on your first attempt, then refine the best one using feedback or smart regeneration.

Smart Regeneration

Refine your infographic while preserving elements you like.

Lock Controls

Lock specific aspects to keep them unchanged during regeneration:

  • Lock Layout: Preserve the overall composition and product position
  • Lock Colors: Keep the color scheme intact
  • Lock Background: Maintain the background style
  • Lock Text Style: Preserve typography choices

Variation Strength

Control how different the new generation will be:

  • Subtle: Minor refinements, very similar result
  • Moderate: Noticeable changes while keeping essence
  • Significant: More dramatic differences
  • Major: Substantial reinterpretation

Use the feedback text field to describe specific changes:

  • "Make the background darker"
  • "Increase the size of the product"

General Tips

Start Simple

Begin with basic settings and the "Auto" style. Add complexity only after you understand what works for your products.

Use High-Quality Images

Better input images lead to better results. Use well-lit photos with clean backgrounds when possible.

Iterate Quickly

Generate 4 variations initially, then refine the best one. Don't spend too long perfecting settings upfront.

Save Templates

Once you find settings that work, save them as a template. This ensures consistency across your product line.

Use Feedback

The feedback field is powerful. Describe exactly what you want changed in natural language.

Match Your Brand

Add your brand colors and use consistent style settings across all products for cohesive marketing materials.

Writing Effective Features

Do's

  • Be specific: "40-hour battery life" not "long battery"
  • Include benefits: "IPX7 waterproof for worry-free workouts"
  • Use numbers: "30W fast charging" is more impactful than "fast charging"
  • Highlight unique features first
  • Match your target audience's language

Don'ts

  • Avoid vague terms: "good quality", "nice design"
  • Don't repeat similar features
  • Skip obvious features (e.g., "comes in a box")
  • Avoid technical jargon unless targeting technical audiences
  • Don't add too many features (6 is usually the sweet spot)

Examples

Weak

  • Good battery
  • Nice sound
  • Comfortable

Strong

  • 40-hour battery with quick charge
  • Hi-Res Audio certified with 40mm drivers
  • Memory foam cushions for all-day comfort
41

Image Quality Tips

Product Photos

  • Use images at least 1000x1000 pixels
  • White or transparent backgrounds work best
  • Ensure product is well-lit and in focus
  • Avoid watermarks or text on product images
  • Show the product at a clear angle

Maximizing Output Quality

  • Use GPT-4o or Flux for highest quality
  • Lower visual density for cleaner results
  • Specify negative prompts to avoid unwanted elements
  • Use style references for consistent branding

Negative Prompts

Use the negative prompt field to specify what to avoid:

  • "blurry, low quality, distorted text"
  • "cluttered, busy, overwhelming"
  • "cartoonish, unrealistic"
  • "watermark, signature, logo"

Platform-Specific Tips

Amazon

  • Use pure white background (required for main images)
  • 1:1 aspect ratio, minimum 1000x1000px
  • Product should fill 85% of the frame
  • Use "Light Background" style preset

Instagram

  • 1:1 for feed posts, 9:16 for Stories
  • High visual density works well
  • Bold colors and gradients perform better
  • Keep text minimal for cleaner look

Shopify/E-commerce

  • Create consistent style across product line
  • Generate complementary images for galleries
  • Use marketing copy for descriptions
  • Save brand templates for consistency

Facebook

  • 16:9 aspect ratio for posts
  • Less visual density, focus on key features
  • Bold headlines that are readable on mobile
  • Use the Facebook social caption from Marketing Copy

Troubleshooting

API Key Error (401)

Cause: Invalid or expired API key

Solution: Check your OpenRouter API key is correct and has available credits

Rate Limit Error (429)

Cause: Too many requests in a short time

Solution: Wait a few minutes and try again, or upgrade your OpenRouter plan

Model Error (500/502/503)

Cause: AI model temporarily unavailable

Solution: Try a different AI model, or wait and retry

Image Not Generating

Cause: Model doesn't support image generation

Solution: Select a model that supports image output (Gemini, GPT-4o, Flux)

Distorted Text in Output

Cause: AI models can struggle with text rendering

Solution: Use GPT-4o for better text, or reduce the number of features

Product Looks Different

Cause: AI interpretation varies

Solution: Upload a clearer product image, increase style reference strength, or use Smart Regeneration with locks

History Not Saving

Cause: Local storage full or disabled

Solution: Clear browser data or check privacy settings

Slow Generation

Cause: Large images, multiple variations, or server load

Solution: Use Gemini Flash for faster results, reduce variations, try during off-peak hours
