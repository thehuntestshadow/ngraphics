# IDEAS.md

Unvalidated ideas parking lot for HEFAISTOS. These are possibilities to explore, not commitments. Ideas graduate to [BACKLOG.md](BACKLOG.md) after validation.

---

## Validation Criteria

Before moving an idea to the backlog, it should pass these checks:

1. **User demand** - Have users actually requested this?
2. **Problem clarity** - What specific problem does it solve?
3. **Effort/value** - Is the ROI worth it?
4. **Fits vision** - Does it align with our core principles?
5. **Technical feasibility** - Can we actually build it?

## Won't Do (Anti-Goals)

These are explicitly out of scope:

| Anti-Goal | Reason |
|-----------|--------|
| Full photo editor | Use Photoshop/Figma - we generate, not edit |
| Video timeline editing | We generate clips, not edit them |
| Social media posting | Use Buffer/Hootsuite - we create, not post |
| Inventory management | Not our domain |
| Complex animations | Short AI clips only, no timeline |
| Fragmented tools | Consolidate into unified studios |
| Real-time chat support | Async support only at current scale |
| On-device AI | Cloud-first for quality and updates |
