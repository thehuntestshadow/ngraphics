# Gallery Assets

Place curated demo images here. Use the naming convention:

## Landing Page Showcase (3x2 grid)
- showcase-1.jpg through showcase-6.jpg

## Landing Page Gallery Preview (horizontal scroll)
- preview-1.jpg through preview-6.jpg

## Gallery Page (filtered by studio)
- infographic-1.jpg, infographic-2.jpg, etc.
- model-1.jpg, model-2.jpg, etc.
- lifestyle-1.jpg, lifestyle-2.jpg, etc.
- bundle-1.jpg, etc.
- social-1.jpg, social-2.jpg, etc.
- packaging-1.jpg, etc.
- background-1.jpg, etc.
- comparison-1.jpg, etc.

## Recommended dimensions
- Showcase images: 800x800px (square)
- Preview images: 560x700px (4:5 ratio)
- Gallery images: 800x1000px (4:5 ratio)
